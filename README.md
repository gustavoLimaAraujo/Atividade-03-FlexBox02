# Atividade 03 - Flexbox 02

Este repositório contém uma atividade prática de desenvolvimento web focada no domínio do **CSS Flexbox**, especificamente na propriedade `justify-content`. O objetivo é entender como alinhar e distribuir elementos dentro de um container ao longo do eixo principal (horizontal, por padrão).

## Sobre o Projeto

A atividade consiste em organizar seis containers diferentes, aplicando as variações da propriedade `justify-content` para atingir layouts específicos.

No arquivo `estilo.css`, foram explorados os seguintes valores para a propriedade `justify-content`:

| Valor | Descrição |
| :--- | :--- |
| `flex-start` | Os itens são alinhados junto à borda de início de acordo com qual for a flex-direction do elemento pai. |
| `flex-end` |  Os ítens são alinhados junto à borda final (end) de acordo com qual for a flex-direction do elemento pai |
| `center` | Os ítens são centralizados na linha. |
| `space-between` | Os ítens são distribuídos de forma igual ao longo da linha; o primeiro ítem "colado" à borda inicial da linha, o último "colado" à borda final da linha. |
| `space-around` | Os ítens são distribuídos na linha com o mesmo espaçamento entre eles, mas o primeiro e o último elemento tem metade de espaçemento entre ele e a borda. |
| `space-evenly` | Os ítens são distribuídos de forma que o espaçamento entre quaisquer itens seja igual.
